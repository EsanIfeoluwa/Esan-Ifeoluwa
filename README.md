# REGISTER Form Project

This is a simple HTML form project that collects user information such as name, email, password, and gender.

## 📌 Features
- User registration form
- Input validation using HTML attributes (required fields)
- Dropdown selection (gender)
- Clean and simple layout

## 🛠️ Technologies Used
- HTML5

## 📂 Project Structure
- Register.html → Main form page

## 🚀 How to Use
1. Download or clone the repository
2. Open `Register.html` in any web browser
3. Fill in the form and click submit


## 🎯 Purpose
This project is for learning how HTML forms work and how user input is collected on websites.
